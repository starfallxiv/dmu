# Ultimate Dancing Mad — Phase 4 Notes

> English raid notes based on the Yan Flash Phase 4 guide for 絶妖星乱舞 / Ultimate Dancing Mad.  
> Source: https://yan-flash.com/ultimate/yosei-ranbu#ult-phase-4  
> Last checked: 2026-07-07

## Overview

Phase 4 is mostly a personal debuff-resolution phase.

Neo Exdeath uses **Grand Cross** multiple times to assign debuffs. Chaos adds either **Fire** or **Tsunami**, giving the party **Chaotic Fire** or **Chaotic Water**. Kefka then marks mechanics as **True** or **Lie**, and the party resolves the stored mechanics later.

The important thing is not memorizing the whole phase as one giant blob. Track **your own debuffs**, track whether each stored mechanic is **True** or **Lie**, and move to the correct position when each mechanic resolves.

## Core True / Lie Rules

| Mechanic | True | Lie |
|---|---:|---:|
| **Cursed Shriek** / 呪詛の叫声 | Do **not** look | Look |
| **Forked Lightning** | Solo spread AoE | Stack |
| **Water Compression** | Stack | Solo spread AoE |
| **Acceleration Bomb** | Stop moving | Keep moving |
| **Chaotic Water** | Stay center / do not move, resolves as donut | Dodge eruptions |
| **Chaotic Fire** | Dodge eruptions | Stay center / do not move, resolves as donut |

### Notes

- **Cursed Shriek** appears in Grand Cross 1 and Grand Cross 2. The timers are fixed, so the party should remember whether the first and second Shriek were True or Lie.
- **Forked Lightning** and **Water Compression** are mutually exclusive. You should not receive both at the same time.
- **Acceleration Bomb** is assigned during either Grand Cross 1 or Grand Cross 2.
- **Chaotic Fire / Water** are assigned to everyone.

## Wound / Color Rules

| Debuff | How to resolve |
|---|---|
| **Dead's Wound** / 死者の傷 | Same color makes you nearly die; opposite color is safe-ish. |
| **Living Wound** / 生者の傷 | Same color makes you nearly die; opposite color is safe-ish. |
| **Allagan Field** | Avoid taking damage. Use the **opposite color** hit. |
| **Beyond Death** / 死の超越 | You must be brought to critical HP. Use the **same color** hit. |

### Practical shortcut for Grand Cross 3

Before checking the later **Void Flood / 無の氾濫** truth value:

- **Allagan Field** = take **opposite color**.
- **Beyond Death** = take **same color**.

Then check **Void Flood**:

| Debuff | Void Flood True | Void Flood Lie |
|---|---:|---:|
| **Allagan Field** | Opposite color | Same color |
| **Beyond Death** | Same color | Opposite color |

In other words: decide your baseline from Allagan/Beyond Death, then flip it only if Void Flood is a Lie.

## Timeline

### 1. Teasing Soul / おちょくりソウル

Kefka and Neo Exdeath appear on the outside of the arena.

### 2. Riddle Magic / なぞなぞマジック

Watch and remember which effects are **True** or **Lie**. These determine how the later mechanics resolve.

### 3. Grand Cross 1

![Grand Cross 1 debuff assignment diagram](https://yan-flash.com/api/uploads/1781181286165-2f3570e3.webp)


Debuffs assigned:

- **Acceleration Bomb**
- **Forked Lightning**
- **Water Compression**
- **Cursed Shriek**, early timing

Resolve the assigned mechanics later according to the True/Lie rules.

### 4. Tsunami or Fire 1

![Tsunami or Fire 1 debuff assignment diagram](https://yan-flash.com/api/uploads/1781181305506-1671d839.webp)


Everyone receives one of:

- **Chaotic Water**
- **Chaotic Fire**

Remember whether the assigned Fire/Water mechanic is True or Lie.

### 5. Grand Cross 2

![Grand Cross 2 debuff assignment diagram](https://yan-flash.com/api/uploads/1781205479283-6059fcf2.webp)


Debuffs assigned:

- **Acceleration Bomb**
- **Forked Lightning**
- **Water Compression**
- **Cursed Shriek**, late timing

Again, store the True/Lie information for later resolution.

### 6. Tsunami or Fire 2

![Tsunami or Fire 2 debuff assignment diagram](https://yan-flash.com/api/uploads/1781181338740-09563e46.webp)


Everyone receives another **Chaotic Water** or **Chaotic Fire** assignment.

### 7. Riddle Magic

Another True/Lie check happens before the third Grand Cross sequence.

### 8. Grand Cross 3

![Grand Cross 3 debuff assignment diagram](https://yan-flash.com/api/uploads/1781181352692-30c9e613.webp)


Debuffs assigned:

- **Living Wound** or **Dead's Wound**
- **Allagan Field** or **Beyond Death**

When resolving this section, use the color shortcut:

- Allagan Field wants **opposite color**, unless Void Flood lies.
- Beyond Death wants **same color**, unless Void Flood lies.

![Grand Cross 3 color resolution diagram](https://yan-flash.com/api/uploads/1783093630380-f6a8b58c.webp)

For the next mechanic, use the arena halves like this:

| Group | Arena half |
|---|---|
| Tanks + Healers | North half |
| DPS | South half |

### 9. Magic Charge

Remember whether the later **Thundaga** and **Blizzaga** tells are True or Lie. This information is reflected in **Magic Out**.

For each stored spell:

| Stored spell | Magic Out | Final result |
|---|---|---|
| True | True | True |
| True | Lie | Lie |
| Lie | True | Lie |
| Lie | Lie | True |

Shortcut: **Magic Out acts like an equality check.** If the stored spell and Magic Out match, the result is True. If they differ, the result is Lie.

## Resolution Sections

### Water Compression + Forked Lightning + Acceleration Bomb

![Water Compression, Forked Lightning, and Acceleration Bomb resolution diagram](https://yan-flash.com/api/uploads/1783093550291-70072f5f.webp)


Spread using **Neo Exdeath's beam** as the orientation reference.

- Solo AoE players spread somewhere safe where they will not clip anyone.
- Stack players group correctly.
- Acceleration Bomb players either stop or keep moving depending on True/Lie.

### Thundaga + Cursed Shriek

![Thundaga and Cursed Shriek resolution diagram](https://yan-flash.com/api/uploads/1781188459365-9fce5986.webp)


- Shriek players gather in the middle.
- Resolve based on whether the relevant Grand Cross Shriek was True or Lie.
- Also remember whether this **Thundaga** was True or Lie, because it affects the later Magic Out.

### Doki Doki Ultima + Chaotic Fire / Chaotic Water

![Doki Doki Ultima with Chaotic Fire or Chaotic Water resolution diagram](https://yan-flash.com/api/uploads/1781188935493-da50ef37.webp)


- **Doki Doki Ultima** is a raidwide.
- **Chaotic Fire / Water** resolves as either eruptions or a donut depending on True/Lie.

### Blizzaga + Water Compression + Forked Lightning + Acceleration Bomb

![Blizzaga with Water Compression, Forked Lightning, and Acceleration Bomb resolution diagram](https://yan-flash.com/api/uploads/1781205832777-12d234b9.webp)


Resolve the Blizzaga movement together with the stored Water/Forked/Acceleration mechanics.

Also remember whether this **Blizzaga** was True or Lie for Magic Out.

### Cursed Shriek

![Cursed Shriek resolution diagram](https://yan-flash.com/api/uploads/1781189992102-82fe2540.webp)


Resolve the remaining Shriek using the stored True/Lie call:

- True = do not look.
- Lie = look.

### Magic Out + Chaotic Fire / Chaotic Water

![Magic Out with Chaotic Fire or Chaotic Water resolution diagram](https://yan-flash.com/api/uploads/1781190552829-90af35ee.webp)


Use the **Magic Charge** equality rule to determine whether the final Thundaga/Blizzaga result is True or Lie, then resolve the final Fire/Water mechanic.

### Doki Doki Ultima

Another raidwide. Push the boss to **24.9% or lower** to enter the final phase.

## Mitigation Notes

The source guide recommends shielding every **Grand Cross**. Barriers for the Black Hole sections are also helpful if MP allows, but Grand Cross shielding is the priority.

Example mitigation ideas from the source include:

| Point | Example mitigation |
|---|---|
| Grand Cross 1 | Healer shields, Temperance, Sacred Soil / Fey Illumination, Kerachole / Panhaima, 90s party mitigation |
| Tsunami / Fire 1 | Healer shields, Temperance, Sacred Soil / Fey Illumination, Kerachole / Panhaima, 90s mitigation |
| Grand Cross 2 | Healer shields, Temperance, Expedient / Fey Illumination / Seraph tools, Holos / Panhaima, 90s mitigation |
| Tsunami / Fire 2 | Healer shields plus extra healer/tank/ranged tools as available |
| Grand Cross 3 | Strong remaining healer tools plus 90s mitigation |

Treat this as a planning template, not gospel. Your party comp and cooldown drift may change the exact assignments.

## Suggested Echo Macros

These are translated callout-style macros. Adjust wording to your static's preference.

```txt
/echo GC1 Shriek: DON'T LOOK
/echo GC1 Shriek: LOOK
/echo GC2 Shriek: DON'T LOOK
/echo GC2 Shriek: LOOK
/echo STACK
/echo SPREAD
/echo STOP
/echo MOVE
/echo ERUPTIONS: FIRE
/echo ERUPTIONS: WATER
/echo Thundaga: LIE
/echo Blizzaga: LIE
```

### When to press each macro

| Situation | Macro |
|---|---|
| GC1 Shriek is True | `GC1 Shriek: DON'T LOOK` |
| GC1 Shriek is Lie | `GC1 Shriek: LOOK` |
| GC2 Shriek is True | `GC2 Shriek: DON'T LOOK` |
| GC2 Shriek is Lie | `GC2 Shriek: LOOK` |
| Forked Lightning is Lie and Water Compression is True | `STACK` |
| Forked Lightning is True and Water Compression is Lie | `SPREAD` |
| Acceleration Bomb is True | `STOP` |
| Acceleration Bomb is Lie | `MOVE` |
| Fire is True | `ERUPTIONS: FIRE` |
| Tsunami / Water is Lie | `ERUPTIONS: WATER` |
| During Magic Charge, Thundaga is Lie | `Thundaga: LIE` |
| During Magic Charge, Blizzaga is Lie | `Blizzaga: LIE` |

## Fast Mental Model

1. **Grand Cross 1/2:** record Shriek, stack/spread, and Acceleration Bomb.
2. **Fire/Water:** record whether you get eruptions or donut.
3. **Grand Cross 3:** Allagan = opposite color, Beyond Death = same color; flip only if Void Flood lies.
4. **Magic Charge:** Magic Out is an equality check.
5. **Do not greed the memory game.** The phase is trying to fry your RAM like Chrome with 37 tabs open.
